# RiotAPI를 활용한 전적검색 사이트
#### nodejs를 활용한 리그 오브 레전드 전적 검색 사이트
#### 본 프로젝트는 학습용입니다.
#### 관련문서 
[RiotAPI](https://developer.riotgames.com/)
[ddrgon](https://developer.riotgames.com/docs/lol)
[Expressjs](https://expressjs.com/en/guide/routing.html)
## 사용방법
### Riot API를 넣어준다.
### npm 모듈을 설치 해준다.
```
npm install 
```
### 실행 방법
```
npm start
```
or
```
node server.js
```
localhost로 접속.
